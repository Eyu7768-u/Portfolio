A minimalist portfolio project built step-by-step, combining clean development practices with a subtle anime-inspired theme.

---

## 📌 Project Overview

This project showcases a structured workflow using Git and GitHub, inspired by precision, growth, and continuous improvement—just like an anime character leveling up.

* 🌿 **Version 1:** Static website using HTML & CSS
* ⚡ **Version 2:** JavaScript features added via Pull Request

---

## 🚀 Features

### 🌱 Version 1 — Foundation Arc (HTML & CSS)

* Semantic and clean HTML structure
* Minimalist, aesthetic design
* Responsive layout
* Sections: About, Skills, Projects, Contact

### ⚡ Version 2 — Power-Up Arc (JavaScript)

* Interactive UI behavior
* DOM manipulation
* Improved user experience

---

## 🔀 Git Workflow

A clear and disciplined workflow:

1. 🌿 Created `version1` branch

   * Built UI using HTML & CSS
   * Committed step-by-step

2. ⚡ Created `version2` branch

   * Added JavaScript functionality

3. 🔥 Opened a Pull Request

   * Merged changes into main branch
